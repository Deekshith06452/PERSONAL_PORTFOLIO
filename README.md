# Deekshith Gowda S — Personal Developer Portfolio

A premium, dark-themed, responsive personal developer portfolio website designed to showcase my projects, skills, education, and certified achievements.

Live Site: [https://github.com/Deekshith06452/PERSONAL_PORTFOLIO](https://github.com/Deekshith06452/PERSONAL_PORTFOLIO)

---

## 🚀 Features

- **Interactive Hero Section**: Features a dynamic typing animation with Typed.js.
- **Featured Projects**: Showcase of my highlighted full-stack and frontend projects, including:
  - **InstantFood**: Full Stack Food Delivery Platform (includes an embedded interactive demo video modal).
  - **User Registration System**: Full Stack Java EE application.
  - **Myntra Clone**: Static frontend clone with full responsive UI.
- **Achievements & Certifications**: A curated, interactive grid displaying my verified certifications that link to the respective PDF certificates:
  - **Complete HTML** (KnowledgeGate)
  - **Parallel Computing** (JuliaAcademy)
  - **Full Stack Web Development** (Tap Academy Internship)
- **Interactive Tech Stack Marquee**: A custom CSS continuous-scrolling marquee showing languages, frameworks, and tools.
- **Timeline/Journey Tracker**: Beautifully organized timelines detailing my internship experience and educational milestones.
- **Contact Form**: Interactive contact form with focus states and responsive success/error toast notifications.
- **Modern UI Elements**: Custom glassmorphism, responsive menus, glowing neon accents, and smooth scroll animations (using AOS).

---

## 🛠️ Tech Stack & Libraries Used

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Animations**: AOS (Animate on Scroll), Typed.js
- **Icons**: Font Awesome 6 Free, Devicon
- **Typography**: Google Fonts (Poppins, Inter, Outfit)

## 💻 Getting Started Locally

To run the project on your local machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/Deekshith06452/PERSONAL_PORTFOLIO.git
   ```
2. Navigate into the project folder:
   ```bash
   cd PERSONAL_PORTFOLIO/Portfolio
   ```
3. Open `index.html` directly in your web browser (or run with a Live Server extension in VS Code).
